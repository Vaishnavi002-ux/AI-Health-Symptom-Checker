🤖 Agentic AI Workflow

1
User Enters Symptoms
Natural language input via chat UI. Voice input supported via Web Speech API.
2
Emergency Detection
40+ emergency keywords scanned. Immediate escalation if match found — no AI call made.
3
Information Assessment
Heuristic check for temporal context, severity, and length. Decides if follow-up is needed.
4
Follow-up Questions (IBM Granite)
Up to 3 targeted follow-up questions generated by Granite. Avoids repeating questions already asked.
5
Structured Prompt Construction
Full conversation history + patient context assembled into a rigidly structured prompt.
6
IBM Granite Inference
Sent to IBM watsonx.ai REST API with IAM bearer token (cached with 5-min refresh buffer).
7
Response Parsing
Regex-based section extraction: Conditions, Urgency, Reason, Home Care, Prevention, Consult, Disclaimer.
8
Structured Results Display
Urgency banner, conditions, care tips, disclaimer rendered. Downloadable JSON report.

🚀 Quick Setup

# 1. Copy environment variables
cp .env.example .env
# Add your IBM credentials to .env

# 2. Start backend
cd backend
python -m venv venv
venv\Scripts\activate        # Windows
pip install -r requirements.txt
python app.py                # http://localhost:5000

# 3. Start frontend
cd frontend
npm install
npm start                    # http://localhost:3000
